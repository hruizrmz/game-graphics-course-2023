My Devices and their GPU

Laptop

Model: Dell G5 SE 5505
CPU: AMD Ryzen 4600H (6 Cores)
  1. Is it integrated or dedicated?
    2.  The GPU is labeled as "AMD Radeon Graphics", but it's the mobile Radeon RX Vega 6. It comes integrated with the AMD CPU. I can run most games, but the performance varies.

  2. What kind of graphic APIs does it support?
    3.  As it's a Windows and AMD device, it supports a couple of APIs:
    • DirectX 12_1 (Windows/Microsoft product)
    • OpenGL 4.6
    • OpenCL 1.2
    • Vulkan 1.2 (AMD product)

  3. What kind of shading languages does it supports?
    4.  I wasn't able to find anything concrete, but I believe the DirectX shader model is 6.7, so it supports HLSL.

Phone

Model: iPhone SE 2nd Generation
SoC: A13 Bionic chip (6 Cores)
  1. Is it integrated or dedicated?
    2.  All iPhone processors come integrated in the Apple chips. This one has a "4-core GPU". Graphic intensive games perform poorly, they overheat my phone, and they quickly drain its battery.

  2. What kind of graphic APIs does it support?
    3.  Since it's an Apple product, it supports the Metal API.

  3. What kind of shading languages does it supports?
    4.  Likewise, the shading language it uses would be MSL.

Gaming Console

Model: Nintendo Switch
SoC: Nvidia Custom Tegra processor
  1. Is it integrated or dedicated?
    2.  This is another chip that handles all processes. It was not officially revealed by Nintendo, but the Switch's is known to be an integrated Nvidia Tegra X1. It has 256 Maxwell-based cores.

  2. What kind of graphic APIs does it support?
    3.  Since the Switch is a multi-developer platform, it supports a range of APIs.
    • DirectX 11.2 (Xbox and PC Ports)
    • OpenGL 4.5
    • CUDA 6.0 (NVIDIA product)

  3. What kind of shading languages does it supports?
    4.  Given the APIs, it supports GLSL and HLSL. It also supports C, C++, and Fortran for handling CUDA.
